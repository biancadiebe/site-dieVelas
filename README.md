# 🕯️ Die Velas

Uma aplicação web desenvolvida para a **Die Velas**, uma marca de velas artesanais veganas. O projeto foi criado com o objetivo de oferecer uma vitrine virtual para os produtos da marca, proporcionando uma experiência moderna, intuitiva e responsiva para os clientes.

Além de atender a uma necessidade real, este projeto também representa a aplicação prática dos conhecimentos adquiridos em desenvolvimento Front-end, utilizando tecnologias modernas e boas práticas de programação.

## Status do projeto

Este projeto está em desenvolvimento e continua recebendo melhorias e novas funcionalidades. 

As próximas atualizações incluem melhorias na interface, otimizações de desempenho, novas funcionalidades para o catálogo e aprimoramentos voltados à usabilidade e acessibilidade.

## Tecnologias utilizadas

* Next.js
* React
* TypeScript
* CSS
* Git
* GitHub

O projeto foi organizado utilizando a estrutura do **Next.js**, priorizando a separação dos componentes e facilitando futuras manutenções e expansões.

## Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/biancadiebe/site-dieVelas.git
```

Acesse a pasta do projeto:

```bash
cd site-dieVelas
```

Instale as dependências:

```bash
npm install
```

Execute a aplicação:

```bash
npm run dev
```

Depois, acesse:

```text
http://localhost:3000
```

## Objetivos do projeto

Este projeto foi desenvolvido com o propósito de:

* Criar uma aplicação para um cliente real;
* Praticar o desenvolvimento com React e Next.js;
* Aplicar conceitos de componentização e organização de código;
* Desenvolver interfaces responsivas e de fácil navegação;
* Consolidar conhecimentos em TypeScript e desenvolvimento Front-end;
* Compor meu portfólio com um projeto completo e em constante evolução.
