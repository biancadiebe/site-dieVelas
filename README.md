# 🕯️ DiêVelas

> ⚠️ Projeto em desenvolvimento

E-commerce de velas artesanais e veganas, desenvolvido como projeto pessoal.

## Objetivo

Desenvolver habilidades práticas voltadas ao desenvolvimento web.

## Tecnologias

- Next.js
- React
- TypeScript
- CSS Modules
- React Icons

## Como rodar o projeto?

npm install
npm run dev


Acesse: http://localhost:3000
